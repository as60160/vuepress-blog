# Vue.js 學習筆記

# 前言

建立這個 「Vue.js 學習筆記」 的目的有二：

  1. 練習 VuePress
  2. 便於回顧 「Vue.js 手牽手」的影音筆記

紀錄在 HackMD 上的筆記我沒有拆開，這個系列的筆記全都放在同一篇，於是原本的 HackMD 筆記共有四千多行，每次點開都會很卡... 藉著練習 VuePress 的機會，便將裡面的筆記依單元拆分，備份到這裡來。

這個系列的教學，我並非一口氣看完，而是斷斷續續看的，期間經歷了 Vue 的改版，因此使用的版本、語法，以及書寫上的習慣都有變化。

其他的筆記也許會在更加成熟之後，另外建立新的 VuePress 專案紀錄並發表，這部分就之後看情況進行。
